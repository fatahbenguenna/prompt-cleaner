Nettoie un texte de ses données personnelles avant de le partager — 100 % hors-ligne : rien ne quitte ce navigateur.

Aucune config chargée
Légende : remplacé suspect non remplacé (à vérifier)
Prêt — chargez un fichier de configuration puis collez votre texte. Astuce : vous pouvez aussi déposer votre fichier .cfg n'importe où sur la page.

Format du fichier de configuration

Fichier texte (UTF-8), une règle par ligne au format « motclé : remplacement ». Au nettoyage, chaque occurrence du motclé est remplacée par la valeur de droite.

# Les lignes vides et celles commençant par # sont ignorées.
# Une règle par ligne, au format : motclé : remplacement

google : mon-entreprise
fb44ja8k : nom-user
myApp : nom-application

# La casse des mots-clés est ignorée (Google = google = GOOGLE).
# Seul le premier « : » sépare : la valeur peut en contenir.
portail : https://intranet.exemple:8080/accueil
  • En cas de mot-clé en double, la dernière ligne gagne.
  • Les mots-clés les plus longs sont appliqués en premier.
  • Cochez « mémoriser dans ce navigateur » pour retrouver vos règles à la prochaine visite (stockage local uniquement, jamais envoyé en ligne).
