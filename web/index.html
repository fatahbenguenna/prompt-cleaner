Affiner l'anonymisation

Le nettoyage automatique détecte déjà chemins Windows, e-mails, adresses IP, téléphones, IBAN, cartes bancaires… Vous pouvez aussi améliorer la précision en chargeant un fichier de règles, ou en éditant le modèle via le point d'interrogation.

Aucune config chargée

Astuce : vous pouvez aussi déposer votre fichier .cfg n'importe où sur la page.

Règles de remplacement

Une règle par ligne au format « motclé : remplacement » — chaque occurrence du motclé (casse ignorée) sera remplacée par la valeur de droite. Modifiez les règles ci-dessous puis cliquez sur Utiliser ces règles : aucun fichier n'est nécessaire.

  • Lignes vides et lignes commençant par # ignorées ; seul le premier « : » sépare.
  • Mot-clé en double : la dernière ligne gagne. Les mots-clés longs sont appliqués en premier.
  • Cochez « mémoriser dans ce navigateur » pour retrouver vos règles à la prochaine visite (stockage local uniquement, jamais envoyé en ligne). « Copier » permet d'enregistrer vos règles dans un fichier .cfg réutilisable.
